这是一个雅思听力单词听写训练应用，主要功能包括：

核心功能
听写练习：显示中文释义，播放英文发音，用户输入英文单词拼写

多章节支持：包含 List 3.1、3.2、3.3 等多个词汇列表

语音朗读：使用浏览器 Web Speech API 自动朗读单词

实时反馈：即时判断拼写正误，错误时显示正确答案

特色功能
随机模式：可打乱单词顺序练习

自动跳转：5秒倒计时，超时自动进入下一题

统计面板：实时显示正确数、准确率、剩余题数

实时日志：记录每次答题的时间、结果和拼写情况

进度追踪：顶部进度条和百分比显示

技术栈
纯前端单页应用（HTML + JavaScript）

Tailwind CSS 样式框架

jQuery 操作 DOM

Web Speech API 语音合成
